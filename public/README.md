# Public Folder

Ide tedd a saját videó és zene fájlokat:

- `video.mp4` - Background videó (vagy más formátum: .webm, .mov)
- `music.mp3` - Háttérzene (vagy más formátum: .wav, .ogg)
- `video-poster.jpg` - Videó poster kép (opcionális)

Ha nincs itt fájl, akkor az App.jsx-ben beállított online URL-eket használja.
